# LAUNCH_PLAN

## Launch Phase
- Phase: MVP trust-first launch (Argentina)
- Window: 2026-05-11 to 2026-05-31

## Success Metrics
- >= 25% of completed hires leave a verified review.
- <= 15% weekly drop-off from request to hire completion.
- >= 30% first-hire conversion for new customer cohorts.
- <= 3% moderation SLA breach rate.

## Campaign Tracks
1. Trust Proof Track
- Goal: Increase qualified demand with credibility assets.
- Channels: SEO landing pages, testimonial snippets, email nurture.
- KPI: CTR to provider profiles >= 4.5%.

2. Provider Activation Track
- Goal: Improve supply quality and review coverage.
- Channels: Onboarding email sequence, WhatsApp assist, sales outreach.
- KPI: Provider profile completion >= 70%.

3. Review Integrity Track
- Goal: Grow review volume without fraud risk.
- Channels: In-product prompts + post-service follow-up emails.
- KPI: Verified review rate >= 25%, fraud-flag ratio <= 2%.

## Owners
- Content-writer: trust assets + LP copy.
- Growth-strategist: channel execution + CAC monitoring.
- Researcher: weekly cohort readout + objection mining.
- Sales-representative: provider objections and close scripts.

## Current Risks
- Lifecycle/platform queue drift can delay execution handoffs.
- Telemetry evidence gate must stay green before scale decisions.
